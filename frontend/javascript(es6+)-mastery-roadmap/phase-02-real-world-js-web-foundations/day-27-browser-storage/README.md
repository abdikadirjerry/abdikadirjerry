# Day 27 — Browser Storage App

## Features

- Save user name (localStorage)
- Save temporary message (sessionStorage)
- Todo list with add/delete (localStorage)
- Dark/Light theme switcher (localStorage)

## Concepts Used

- localStorage
- sessionStorage
- JSON.stringify()
- JSON.parse()
- DOM manipulation
- Modular JavaScript

## Project Structure

- index.html → UI
- style.css → Styling
- app.js → Main logic
- storage.js → Storage functions
- utils.js → Helper functions

## How to Run

1. Open index.html in browser
2. Interact with the UI
3. Refresh to test persistence
